# Live Code Editor üìù

A minimalist, high-performance, in-browser code editor built with React, Vite, and the powerful Monaco Editor. This application provides a seamless development experience with a real-time preview pane, allowing you to write and visualize HTML, CSS, and JavaScript code instantly.

## Features

-   **Real-time Preview**: Instantly see the output of your HTML, CSS, and JavaScript code as you type.
-   **Resizable Panes**: Built with `react-split`, the editor and preview panes can be resized for a comfortable and customized workflow.
-   **Professional Code Editor**: Powered by `@monaco-editor/react`, the same editor that powers VS Code, offering syntax highlighting, and intelligent code completion.
-   **Modern Tech Stack**: Leverages the speed of Vite for a lightning-fast development server and optimized builds.
-   **Styled with Tailwind CSS**: A utility-first CSS framework for rapidly building a clean and responsive user interface.

## Usage

To get started with the editor, follow these steps:

1.  Clone the repository and install the dependencies:
    ```bash
    git clone https://github.com/your-username/code-editor.git
    cd code-editor
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```
3.  Open your browser and navigate to the local URL provided (usually `http://localhost:5173`).

You will be presented with a split-screen interface. The left side contains the code editor with tabs for HTML, CSS, and JavaScript, while the right side displays the live-rendered output of your code.

-   Write your HTML structure in the `HTML` tab.
-   Add styles in the `CSS` tab.
-   Implement logic and interactivity in the `JavaScript` tab.

The preview pane on the right will update automatically as you make changes.
